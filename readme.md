# Résumé :

ce projet est une implémentation graphique des tours de Hanoi

son propos est avant tout pédagogique, il permet de présenter les solutions optimales pour des piles de 1 à 10 disques.

Si vous avez accès à ce dépot c'est que je vous ai encouragé à analyser ce code.
Faites le et n'hésitez pas à me poser des questions, faire des remarques des critiques ou proposer des améliorations.

# Description

le projet utilise trois fichiers (dont une icône qui est dispensable)

- `liste_hanoi.py` permet de créer une liste des états des trois mats au fil de la résolution et est utilisée pour y "naviguer" facilement avec l'implémentation graphique
- `gui_hanoi.py` est l'implémentation graphique, Je prends le parti de ne pas faire de POO car ici rien ne l'impose vraiment les classes de Widgets de Tkinter sont déjà tout à fait suffisantes.
- `iconeNSI.png` sert d'icone pour la fenêtre Tkinter crée.
